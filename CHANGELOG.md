# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-15

### Added
- Initial release for Strapi v5
- Support for public buckets with permanent URLs
- Support for private buckets with time-limited signed URLs
- Full TypeScript implementation with type definitions
- Node.js 20, 22 support
- Configurable file size validation
- Automatic file path management
- Comprehensive error handling and logging
- Detailed documentation and troubleshooting guide

### Features
- Public bucket mode with permanent public URLs
- Private bucket mode with signed URL generation
- Configurable signed URL expiration time
- Optional directory organization within buckets
- Cross-platform path handling
- Browser and Node.js compatibility

### Security
- No data collection or telemetry
- Environment variable-based credential management
- Secure signed URL generation for private files
- Protection against common vulnerabilities
